This website is a patient portal application which will allow the user to use eHealth Ontario web services. The main feature of the website is a permission management system that allows a patient using the main patient account to set permissions for a caregiver to view thier health records. The patient will have the ability to fine tune permissions down to the individual record level. This means that for each record a patient has, they can turn on or off the permissions to view that page.

Our view will allow viewing of:

  • lab test results
  • diagnostic image test results
  • patient visits to careproviders
  • medications
  • allergies

Caregiver Access Section

  • list of linked caregiver accounts
  • permission settings
